¶变量 😜

  • 定义变量
  • 变量可以被赋值，还可以被重新赋值

¶数据类型

  • 整数类型：byte，short，int，long
  • 浮点类型：float，double
  • 字符类型：char
  • 布尔类型：boolean

¶常量

  • 用final定义
  • 通常全部大写
  • 避免magic number
